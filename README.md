# FindIt

## Project Overview
FindIt is an innovative solution designed to help users quickly locate items in various environments. The project aims to streamline the search process, making it efficient and user-friendly.

## Key Features
- **Real-time tracking**: Allows users to see the location of items in real time.
- **User-friendly interface**: Simple and intuitive design for better accessibility.
- **Search optimization**: Advanced algorithms for faster and more accurate item searches.

## Tech Stack
- **Frontend**: React.js, Bootstrap
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Deployment**: Docker, Heroku

## Database Schema
- **Users Table**: Stores user information including name, email, and password.
- **Items Table**: Contains item details such as item_id, name, location, and status.

## Installation Instructions
1. Clone the repository: `git clone https://github.com/ananyasingh2307/FindIt.git`
2. Navigate to the project directory: `cd FindIt`
3. Install dependencies: `npm install`
4. Start the application: `npm start`
